
### To use
- npm install // to install the needed dependencies
- npm install -g json-server // to install the json server globally
- Enter this on your terminal, json-server --watch data/db.json --port 8000 // the server is fetched at port 8000, if you change this, you will have to change it in the codebase as well
- npm run start


 This project is done by following a NetNinja Tutorial from Youtube on Full React Tutorial.
